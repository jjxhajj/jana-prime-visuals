# Jana Prime Visuals — Portfolio

Freelance creative portfolio for Jana Prime Visuals, based in Vienna.

**Built with:** Next.js 16 · Tailwind CSS · shadcn/ui  
**Deployed on:** Vercel  
**Built using:** Claude Code (end-to-end)

## Pages
- `/` — Home
- `/about` — About
- `/services` — Services (7 services)
- `/case-studies` — Case Studies (3 including meta case study)
- `/contact` — Contact form

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Push to GitHub, connect to Vercel — it deploys automatically.

---

> This portfolio is itself a case study. It was built end-to-end using Claude Code in a single session, captured as the "meta case study" on the Case Studies page.
